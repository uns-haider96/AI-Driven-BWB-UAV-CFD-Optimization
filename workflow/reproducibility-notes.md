# Reproducibility Notes

## What Is Included

This repository includes:

- CAD image of the final geometry
- CFD mesh and contour images
- Fluent simulation report
- Design-space sampling plots
- Surrogate-model comparison plots
- Bayesian optimization convergence figure
- Baseline-vs-optimized result table
- Composite manufacturing images
- Wind tunnel testing images and force/moment summary plot
- IBCAST presentation and participation certificate
- Publication record and DOI information

## What Is Not Included

This public release intentionally excludes:

- Conference paper PDF
- Publisher-formatted journal PDF
- Journal manuscript DOCX
- Raw ANSYS/Fluent case and data files
- Editable CAD/STEP/SolidWorks files
- Unreviewed private drafts
- Uncleaned working scripts

## What Can Be Added Later

To make the project computationally reproducible, the following can be added in future releases after review and approval:

- Cleaned Python scripts for surrogate-model training
- Bayesian optimization notebook
- Input design matrix
- CFD output table
- Processed wind tunnel dataset
- CAD/STEP files approved for public release
- Mesh-independence study
- Force-balance calibration and uncertainty notes

## Public-Release Position

This version is designed as a professional portfolio release based on the available project archive and publication record. It avoids uploading private drafts, large solver files, or publisher-controlled journal/conference files.
