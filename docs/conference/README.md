# Conference Documentation

This folder contains public-safe conference documentation for the IBCAST 2025 presentation.

Included:

- `IBCAST-2025-participation-certificate.pdf`
- `IBCAST-2025-presentation.pptx`

Not included:

- Conference paper PDF
- Publisher-formatted journal PDF
- Journal manuscript DOCX

The conference paper is intentionally excluded from the public repository to avoid copyright, publisher-policy, and proceedings-status issues.

Recommended wording:

> This work was presented at IBCAST 2025. Conference proceedings / IEEE Xplore indexing status should be updated only after official availability is confirmed.
