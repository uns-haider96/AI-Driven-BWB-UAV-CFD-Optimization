# Baseline vs Optimized Design

## Main result

The optimized BWB UAV achieved an approximate **23% improvement** in lift-to-drag ratio compared with the baseline configuration.

| Parameter / Metric | Baseline Design | Optimized Design |
|---|---:|---:|
| Wingspan | 1.0 m | 1.0 m |
| Wing area | 0.75 m² | 0.85 m² |
| Sweep angle | 30° | 28° |
| Taper ratio | 0.50 | 0.48 |
| Wing twist | -2° | -3° |
| Winglet cant / sweep | 15° / 10° | 20° / 15° |
| Lift coefficient, CL | 0.330 | 0.365 |
| Drag coefficient, CD | 0.0335 | 0.0300 |
| Lift-to-drag ratio, L/D | 9.9 | 12.2 |
| Predicted CL,max | 1.1 | 1.3 |
| Empty weight | 2.5 kg estimated | 3.1 kg actual |
| MTOW | 3.5 kg | 4.1 kg |

## Interpretation

The optimized design improved lift generation and reduced drag relative to the baseline. The improvement in L/D supports the project objective of increasing aerodynamic efficiency for low-power UAV operation.
