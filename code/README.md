# Code

This public release is a documentation-first research portfolio. The available project archive contains reports, figures, conference material, publication evidence, and simulation outputs, but not cleaned source-code files.

Future code additions can include:

- Design-space preprocessing scripts
- Surrogate-model training scripts
- Gaussian Process Regression model training
- Bayesian optimization loop
- CFD result post-processing
- Plot-generation notebooks

Do not upload unreviewed scripts or raw data until they are cleaned, documented, and approved for public release.
