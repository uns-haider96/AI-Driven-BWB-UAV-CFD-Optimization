# Project Overview

This project investigates the design and optimization of a one-meter blended-wing-body UAV for low-power, endurance-focused missions.

The BWB configuration was selected because the wing and fuselage merge into a single lifting surface, allowing improved aerodynamic efficiency compared with conventional small UAV layouts. The project focused on improving the aerodynamic design while maintaining practical manufacturability and structural feasibility.

## Main objective

Improve the UAV lift-to-drag ratio and reduce propulsion-power demand through aerodynamic shape optimization.

## Engineering workflow

1. Design a 1-meter BWB UAV geometry.
2. Define geometric design variables.
3. Generate design candidates using Latin Hypercube Sampling.
4. Evaluate candidate designs using CFD.
5. Train surrogate models for rapid performance prediction.
6. Use Bayesian optimization to identify improved designs.
7. Validate the optimized configuration using CFD, structural analysis, prototype fabrication, and wind tunnel testing.

## Key contribution

The project integrates simulation, AI-assisted optimization, fabrication, and experimental assessment in one coherent UAV development workflow.
